# Build-time generation brief: basic-syntax breadth Batch 013

Generate exactly 24 schema-v2 `standard` Markdown exercises for Nabi Markdown:
12 Level 1 italic-emphasis lessons and 12 Level 2 short document rebuilds. This
is an immutable, review-gated Issue #9 batch, never runtime AI output.

This batch follows the basic-syntax scope documented by all four sources named
in Issue #9's 2026-07-19 refinement:

- https://daringfireball.net/projects/markdown/basics
- https://daringfireball.net/projects/markdown/syntax
- https://www.markdownguide.org/basic-syntax/
- https://markdown.kr/guide.php

Those sources agree that single asterisks or underscores create phrase-level
emphasis. Grade the parsed `emphasis` structure only. `*words*`, `_words_`,
combined strong-plus-italic syntax, and real italic nested in a link label are
valid. Bold-only syntax, unmatched or escaped markers, raw HTML, inline or
fenced code lookalikes, and image-alt lookalikes do not satisfy the lesson.
Never compare learner wording, target prose, protected content, vocabulary,
case, spelling, punctuation, or meaning.

Level 1 uses one short familiar phrase, visible instruction, `introduce` mode,
and the `everyday` vocabulary profile. Level 2 implements the revised
curriculum rather than the old recall-only model: rebuild a whole three-block
document that combines one H1, one italic phrase, and either an unordered list,
an ordered list, or a blockquote. Use four genuinely different everyday
variants for each of those three retry families. The Goal is visible, but
learner prose may differ; only the declared Markdown anatomy is graded.

For every candidate, provide canonical, different-prose, case/spelling,
missing, malformed, matched-with-review, equivalent-marker, and adversarial
parser fixtures. Every match check must have direct failing evidence. Freeze
the current known grammar-only boundary that hidden HTML-comment list items
still count as nonempty; do not change existing verdict semantics inside this
content batch.

The four links define syntax scope only. Do not crawl them or any third-party
site for vocabulary. Vocabulary is authored in this build-time session and
must remain common contemporary US English. Images stay deferred until the
existing local-asset and alt-text visual-review gate is satisfied; code blocks,
Setext headings, escapes, and hard breaks remain later breadth candidates and
are not claimed as covered by this batch.

Bind all candidates and fixtures to batch
